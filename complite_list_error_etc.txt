구현내역 / 기능

O : h(왼쪽), j(아래), k(위), l(오른쪽) : 창고지기 조정
O : u(undo) : 최대 5번 할 수 있음
O : r(replay) : 현재 맵을 처음부터 다시 시작(움직임 횟수는 계속 유지)
O : n(new) : 첫 번째 맵부터 다시 시작(움직임 횟수 기록 삭제)
O : e(exit) : 게임 종료. 종료하기 전 필요한 정보 저장해야 함
O : s(save) : 현재 상태 파일에 저장. 파일 이름은 sokoban으로 하고 다음에 다시 게임을 연속해서 할
수 있도록 모든 상태 저장해야 함
O : f(file load) : sokoban 파일에서 저장된 내용을 읽어 save 시점에서부터 이어서 게임하게 함
X : d(display help) : 명령 내용 보여줌
X : t(top) : 게임 순위 보여줌. t 만 입력하면 전체 순위. t 다음 숫자가 오면 해당 맵의 순위
